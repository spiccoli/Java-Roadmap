# Stage 0 — Java Foundations

## Goal
Become fluent with Java syntax and basic OOP.

## Topics

- Primitive types and operators  
- Control flow (if, switch, loops)  
- Arrays and strings  
- Classes and objects  
- Encapsulation  
- Inheritance  
- Polymorphism  
- Interfaces vs abstract classes  
- Basic exception handling  
- Collections basics (List, Set, Map)

## Projects

- CLI calculator  
- Student manager (in-memory CRUD)  
- Simple banking model
